# MKFramework
Custom OpenGL enigne, written on C++
Using libs: GLAD, SDL2, Dear IMGui
Three main modules:
-MKGraphics (OpenGL wrapper, some graphics utils)
-MKCore (windowing, input, events)
-MKUtils (utils)
